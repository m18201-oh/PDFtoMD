# PDFtoMD

PDF 파일을 Markdown 형식으로 변환하는 도구입니다.

## 기능

- PDF 파일을 Markdown(.md) 형식으로 변환
- 텍스트, 제목, 목록 등 문서 구조 보존
- 표(Table) 변환 지원
- 이미지 추출 및 삽입
- 다중 페이지 PDF 처리

## 설치

```bash
git clone https://github.com/your-username/PDFtoMD.git
cd PDFtoMD
```

## 사용법

```bash
# 기본 사용법
python main.py input.pdf

# 출력 파일 지정
python main.py input.pdf -o output.md
```

## 요구 사항

- Python 3.8 이상
